## How to sketch a polar graph
- #### ex.
	- $r=4\sin(3\theta)$
		- make a table
| $\theta$  | R   |
| - | --- |
| 0 | 0   |
| $\frac{\pi}{6}$ | 4   |
| $\frac{\pi}{3}$ | 0   |
| $\frac{\pi}{2}$ | -4  |-
- ###### Area
	- $A=\int _{0}^{\pi}{\frac{r^{2}}{2}} \, d\theta$
		- => $u=3\theta,du=-3d\theta$
			- => $\frac{1}{2}\int _{0}^{\pi} 16\sin^{2}3\theta\, d\theta$
				- $\frac{8}{3}\int _{0}^{3\pi}\sin^{2}u \, du$
					- ###### (using any sin reduction formula)
						- => $4\pi$
